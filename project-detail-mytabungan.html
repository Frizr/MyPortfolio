Project Overview

A Flutter application designed for personal finance planning and tracking, leveraging modern UI patterns and Dart.

Problem

Managing personal finances and tracking daily transactions can be difficult without a dedicated, user-friendly mobile tool.

Goal

The goal is to create a sleek, modern personal finance planner that functions smoothly across mobile operating systems using the Flutter framework.

My Role

I focused on the mobile application development, building the core UI components and logic with Flutter and Dart.

Tech Stack

Flutter 3.11 Dart

Main Features

  • Cross-platform deployment (Android/iOS/Web)
  • Modern UI architecture
  • Financial tracking views
  • Responsive mobile layout

Development Process

Proses pengembangan dimulai dengan perancangan arsitektur UI/UX yang modern. Setelah itu, dilanjutkan dengan implementasi antarmuka menggunakan Flutter, mengonfigurasi rute navigasi, dan menyusun logika aplikasi dasar menggunakan bahasa Dart.

Challenges

Tantangan utamanya adalah memastikan UI aplikasi tetap mulus (smooth) dengan framerate tinggi di berbagai ukuran layar smartphone, serta menjaga agar struktur kode Dart tetap rapi dan mudah di-maintain.

Result

Hasil akhirnya adalah prototipe aplikasi pencatatan keuangan pribadi yang memiliki tampilan modern, responsif, dan memberikan pengalaman pengguna (UX) yang sangat baik di platform mobile.

What I Learned

Melalui proyek ini, saya belajar banyak tentang cara mendesain tata letak UI yang kompleks di Flutter, mengelola widget tree dengan lebih efisien, dan praktik terbaik dalam menstrukturisasi proyek Dart.
