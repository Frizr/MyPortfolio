## ✨ Main Features

- **Responsive Dark Theme UI**: A modern aesthetic with glassmorphism panels and neon accents.
- **Interactive 3D Hero**: Uses WebGL & Three.js to render a responsive, mouse-tracking 3D Icosahedron model in the dashboard.
- **Physics-based Animations**: Implements smooth `IntersectionObserver` scroll reveals and 3D hover-tilt mouse tracking for project/skill cards.
- **Dynamic Navigation**: Sticky navbar with active-state scroll tracking.
- **Comprehensive Sections**: Home, About, Experience, Skills, Services, Projects, Learning Notes, and Contact.
- **Project Detail Pages**: Deep-dive case studies explaining the problem, goal, tech stack, and lessons learned for each repository.
- **Working Contact Form**: Integrated with FormSubmit for direct email routing.

## 🛠️ How to Open Locally

1. Clone this repository: `git clone https://github.com/Frizr/portofolio.git`
2. Open the project folder in your preferred editor (e.g., VS Code).
3. We recommend using the **Live Server** extension to open `index.html` (this ensures the Three.js canvas loads correctly without CORS issues).
4. Or, simply drag and drop `index.html` into your browser.
